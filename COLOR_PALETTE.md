## 📋 Variables CSS

### Uso en HTML/CSS

```html
<!-- Light Mode (por defecto) -->
<div style="color: var(--foreground);">Texto</div>
<div style="background-color: var(--background);">Fondo</div>
<div style="border-color: var(--border);">Borde</div>

<!-- Dark Mode (automático con .dark class) -->
<html class="dark">
  <!-- Las variables CSS se actualizan automáticamente -->
</html>
```

## 🎯 Importación

### En TypeScript/React

```typescript
import { COLOR_PALETTE, TAG_COLOR_PALETTE } from '@/constants/color-palette'
import { DEFAULT_TAG_COLORS, TAG_COLOR_PRESETS } from '@/constants/tag-colors'

// Obtener paleta actual
const isDark = document.documentElement.classList.contains('dark')
const colors = isDark ? COLOR_PALETTE.dark : COLOR_PALETTE.light

// Acceder a colores
const primaryColor = colors.primary        // #0052CC (light) o #3B82F6 (dark)
const backgroundColor = colors.background  // #FFFFFF (light) o #0F172A (dark)

// Tags
const tagColor = DEFAULT_TAG_COLORS.finished  // #10B981
const customColors = TAG_COLOR_PRESETS       // Array de 20 colores
```

### En CSS

```css
/* Usar variables CSS directamente */
.button {
  background-color: var(--primary);
  color: var(--onPrimary);
  border-color: var(--border);
}

/* Responde automáticamente a dark mode */
/* No necesita media queries especiales */
```

### En Tailwind

```jsx
// Los colores de Tailwind se heredan automáticamente
<div className="bg-background text-foreground">
  Contenido
</div>

// Estados
<div className="bg-surface-hover">
  Hover
</div>
```

## 📖 Guía de Uso

### 1. **Componentes Principales**

```jsx
// Botón Primario
<button className="bg-primary text-onPrimary hover:bg-primary-dark">
  Acción Principal
</button>

// Card
<div className="bg-surface border border-border rounded-lg p-4">
  Contenido
</div>

// Sidebar
<nav className="bg-sidebar text-sidebar-foreground">
  Navegación
</nav>
```

### 2. **Textos**

```jsx
// Texto principal
<p className="text-foreground">Texto principal</p>

// Texto secundario
<p className="text-foreground-muted">Texto secundario</p>

// Texto deshabilitado
<p className="text-foreground-light">Texto deshabilitado</p>
```

### 3. **Estados**

```jsx
// Éxito
<div className="bg-green-100 dark:bg-green-900 text-success">
  ✓ Completado
</div>

// Error
<div className="bg-red-100 dark:bg-red-900 text-error">
  ✗ Error
</div>

// Advertencia
<div className="bg-amber-100 dark:bg-amber-900 text-warning">
  ⚠ Advertencia
</div>
```

### 4. **Tags**

```jsx
import { TAG_COLOR_PALETTE } from '@/constants/color-palette'

<span style={{ backgroundColor: '#0052CC', color: '#FFFFFF' }}>
  Tag Azul
</span>

// Con paleta personalizada
TAG_COLOR_PALETTE.custom.map(color => (
  <option key={color} value={color}>
    {color}
  </option>
))
```

### 5. **Transiciones entre modos**

```jsx
// El sistema de tema maneja transiciones automáticamente
// No se necesita código especial

// Para cambiar tema manualmente:
document.documentElement.classList.toggle('dark')
```

## 📏 Especificaciones Técnicas

### Contraste (WCAG AA)

```
Light Mode:
- Texto principal (#0D1B2A) sobre fondo (#FFFFFF):     21:1 ✓
- Texto secundario (#404854) sobre fondo (#FFFFFF):   11:1 ✓
- Botón (#0052CC) sobre fondo (#FFFFFF):              4.5:1 ✓

Dark Mode:
- Texto principal (#F1F5F9) sobre fondo (#0F172A):    16:1 ✓
- Texto secundario (#CBD5E1) sobre fondo (#0F172A):   10.5:1 ✓
- Botón (#3B82F6) sobre fondo (#0F172A):              7.2:1 ✓
```

### Escala de Grises

```
Light: #FFFFFF → #F8F9FC → #F0F2F7 → #D1D5DB → #7A869D → #0D1B2A
Dark:  #F1F5F9 → #CBD5E1 → #94A3B8 → #475569 → #334155 → #0F172A
```
